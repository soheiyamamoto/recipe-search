- 最終更新: 2026-07-05
- 管理: spec-writer(辞書変更時は doc-updater が本書を同期)
- 実装: [../tools/update_recipes.py](../tools/update_recipes.py) /
  配布物: [../src/sanpiryoron_seasonal_recipes.html](../src/sanpiryoron_seasonal_recipes.html)

## 1. 概要

YouTube アップロード一覧 CSV から、レシピ検索帳 HTML 内の
`// ===RECIPES_START===` 〜 `// ===RECIPES_END===` 区間(`const recipes = [...]`)を
再生成する。エンコーディングは入出力とも UTF-8 BOM。

- 対応 CSV 列: `title`, `video_id`, `published_at`, `url`, `description`
- 任意列(手動指定): `season`, `materials`, `exclude` — 値がある行は自動判定より優先
- 並び順: 公開日(`published_at`)降順(新しい動画が上)

## 2. 除外ルール

以下のいずれかに該当する CSV 行はレシピとして反映しない。

| # | ルール | 判定 | カウント |
|---|---|---|---|
| 1 | CSV `exclude` 列に値あり | 手動除外指定 | 企画動画等 |
| 2 | `description` が「↓詳しいレシピは本編」で始まる | ショート版(本編と重複) | ショート版 |
| 3 | タイトルが EXCLUDE_KEYWORDS のいずれかを含む | 企画動画 | 企画動画等 |

※ `title` が空の行も読み飛ばす(カウント外)。

### EXCLUDE_KEYWORDS(企画動画キーワード・10語)

```
生配信 / 密着 / 発売 / 舞台裏 / フェス / サミット / ついて行った /
オープン初日 / 市場で食材探し / 突破ライブ / ありがとう!生配信
```

※「ありがとう!生配信」は「生配信」に包含されるが、明示のため辞書に残している。

## 3. 材料辞書 MATERIAL_DICT(60項目)

検索キーワード(materials 配列)には一致したタグの表記ゆれ全体を登録するため、
「ぶり」「鰤」いずれの入力でもヒットする。
CSV `materials` 列に値がある場合はスペース区切りで分割し、先頭3件をタグ、

## 4. 季節判定仕様

判定の優先順位:

1. **CSV `season` 列** に値がある行はその値(小文字化して使用。`all` = 通年もここで指定)
2. **タイトル内キーワード**(SEASON_KEYWORDS)— spring → summer → autumn → winter の順に
   走査し、最初に一致した季節
3. **公開月**(`published_at`)— 3〜5月 = spring / 6〜8月 = summer /
   9〜11月 = autumn / 12〜2月 = winter
4. 公開日時が解釈できない場合のみ `all`

※「通年(all)」は自動では付与されない。定番料理を通年にしたい場合は
CSV の `season` 列に `all` を指定する。
